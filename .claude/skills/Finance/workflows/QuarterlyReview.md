# QuarterlyReview Workflow

**Comprehensive periodic financial health assessment.**

## Trigger Phrases
- "quarterly financial review"
- "periodic financial checkup"
- "Q1/Q2/Q3/Q4 review"
- "annual financial review"

## Workflow Steps

### Step 1: Determine Review Scope

```
Review Type: [Quarterly / Semi-Annual / Annual]
Period: [Q1 2025 / H1 2025 / 2024]
Areas to Cover:
- [ ] Investment portfolio
- [ ] Retirement accounts
- [ ] Tax planning
- [ ] Budget/cash flow
- [ ] Insurance
- [ ] Estate planning
```

### Step 2: Gather Data

Collect information for each area:

**Investments:**
- Current portfolio positions
- Performance since last review
- Contributions/withdrawals

**Retirement:**
- 401k/IRA balances
- Contribution levels
- Asset allocation

**Taxes:**
- Estimated tax liability
- Tax-loss harvesting done
- Upcoming tax events

**Budget:**
- Income vs expenses
- Savings rate
- Emergency fund status
